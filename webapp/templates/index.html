IPv6 Subnet Planner



This IPv6 Subnet Planner takes the input prefix and subnet size and outputs a "plan" for allocateing IPv6 subnets with a user-specified prefix length. This can be useful for visualizing the subnets and for importing them into other things for more effective use. The putput can be quite large, so be mindful of the server resources as large numbers of prefixes may time out or take for-ev-er.
