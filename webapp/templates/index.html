IPv6 Subnet Planner



This IPv6 Subnet Planner takes the input prefix and subnet size and outputs a json formatted "plan" for allocating IPv6 subnets with a user-specified prefix length. This can be useful for visualizing the subnets and for importing them into other things for more effective use. The output can be quite large, so be mindful of the server resources as large numbers of prefixes may time out or take for-ev-er.
This can also be refererenced by it's API, but again, be mindful of resources. Example API use:
curl -X POST -d "subnet=3ffe:1::/32" -d "prefix=40" https://tools.forwardingplane.net/v6planner/api/subnet
